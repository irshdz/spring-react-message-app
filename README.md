# Spring React Message App

A full-stack messaging application built using Spring Boot and React.

## Tech Stack
- Java
- Spring Boot
- MongoDB
- React / Next.js
- REST APIs
- JWT Authentication

## Features
- User authentication
- Send and receive messages
- RESTful API integration
- Responsive frontend UI

## Architecture
Frontend communicates with backend using REST APIs.
Backend handles authentication, business logic, and database operations.

## Deployment
https://spring-react-message-app.vercel.app

## How to Run

Backend:
1. Open server folder
2. Run Spring Boot application

Frontend:
1. Open client folder
2. Run npm install
3. Run npm start
